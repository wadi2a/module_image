I. Introduction
--------------------------------------------------------------------------------
L'objet de ce document est de vous aider à compiler et executer le module image.

II. Compilation et execution
--------------------------------------------------------------------------------
(A) Sur un PC Windows
---------------------
Pour utiliser et compiler le module image sur un PC qui fonctionne avec Windows, vous devez :
1. Télécharger codeblock 
2. Ouvrir le projet 11512099
3. Utilisé ctrl+F9 pour compiler et executer le projet

(B) Sur un PC Linux 
-------------------
1. Acceder au répertoire du projet 11512099 à l'aide d'un terminal,
2. executer la commande >$ make 11512099 pour executer le petit programme d'image ou >$ make 11512099test pour compiler le test de 	     regression ,
3. Puis vous pouvez executé l'une des deux a l'aide de >$ ./11512099.out ou ./11512099test.out  

III. Description du module
--------------------------------------------------------------------------------
Le module image comporte quelques fonctions pour manipuler des images:
-création
-initialisation
-modifier des pixels
-dessiner un rectangle
-lecture d'une image apartir d'un fichier .ppm
-sauvgarde d'un image dans un fichier de type ppm
-...

--------------------------------------------------------------------------------
tout le dossier de documentation se trouve dans le dossier doc sous le dossier du projet 11512099.
